

# Chat messenger

## Technologies used : Reactjs , streamchat , css , jsx , nodeJs , express

### About working : 
1) User is able to signup 
2) User is able to login 
3) Authentication handling 
4) Chat in a group 
5) Like messages 
6) Share pictures
7) Logout 
8) Compatible for every screen size
